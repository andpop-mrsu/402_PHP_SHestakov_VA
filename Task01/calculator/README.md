# Calculator Game

Консольная игра "Калькулятор" - арифметический тренажер.

## Установка
git clone https://github.com/Baklaniso/calculator.git

cd calculator


# 2. Установить зависимости
composer install

# 3. Сгенерировать автозагрузку
composer dump-autoload -o

# 4. Запустить игру
./bin/calculator
